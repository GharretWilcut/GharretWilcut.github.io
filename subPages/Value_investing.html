Value Investing Score Generator

About the Project

I developed this Value Investing Score Generator to assist investors in evaluating stocks based on key financial metrics. The tool calculates a composite score for each stock by analyzing various factors such as price-to-earnings ratio and debt-to-equity ratio. The goal of the project was to create a systematic approach to identify undervalued stocks that have strong fundamentals, aiding investors in making informed decisions. This project was motivated by my interest in finance and value investing principles. This project was inspired from my Hagan Scholarship with part of it being to learn about investing and personal finance. This project was implemented in Python and Jupyter Notebook. Using libraries such as Pandas for data manipulation and yfinance for fetching stock data made the process efficient and streamlined.

This was a small project that didn't take that long to make but was intersting to me as I was learning about value investing and wanted to make a tool to help me with stock analysis.
