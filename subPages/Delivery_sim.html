Delivery Simulation

About the Project

I made a Delivery Simulation project using Dijkstra's algorithm to optimize delivery routes in a city map. The simulation uses OpenStreetMap data to represent the city layout and road network. This was done to evaluate the efficiency of delivery routes and minimize travel time and fuel consumption. This was because I was interested in logistics and route optimization algorithms. I implemented the simulation in Python and Jupyter Notebook. This project was done as for my Honors Project for Algorithms. One interesting thing I found was when using Dijkstra's algortihm I saw a massive boost in speed using the Heap data structure compared to a simple array implementation.

The simulation allows for testing different numbers of deliveries to see how the algorithm performs under various conditions. Overall, this project provided valuable insights into route optimization and the practical applications of graph algorithms in real-world scenarios.
